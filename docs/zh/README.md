# 软件工程师的工具

_这是我在多年的软件工程师之旅中发现有用或有趣的网站链接、工具和程序的档案。_

此列表的概念基于[awesome](https://github.com/sindresorhus/awesome)和[useful-tools](https://github.com/trolologuy/useful-tools)。

我想创建一个随着我在全栈开发、DevOps、区块链等方面的经验而成长的。
